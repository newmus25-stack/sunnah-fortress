📋 خطوات الإعداد

الخطوة 1: إنشاء حساب Formspree

  1. اذهب إلى https://formspree.io
  2. أنشئ حساباً جديداً أو سجل الدخول
  3. انقر على "Create Project"
  4. أكمل بيانات المشروع

الخطوة 2: إنشاء نموذج بريد

  1. انقر على "Create Form"
  2. اختر نوع النموذج: "Email"
  3. ستحصل على معرف النموذج (مثل: f/abc123def456)

الخطوة 3: تحديث الإعدادات

افتح ملف newsletter-config.js وحدّث الجزء التالي:

const NEWSLETTER_CONFIG = { // استبدل "xyzabcde" برمز Formspree الفعلي FORMSPREE_ID: "f/your-actual-formspree-id", ... };
⚠️ تنبيه: تأكد من استبدال xyzabcde برمز Formspree الفعلي

الخطوة 4: تحديث ملف app.js

في ملف app.js، تحقق من أن دالة handleNewsletterSubmit موجودة وتعمل بشكل صحيح.

// هذا الجزء موجود بالفعل في app.js: const newsletterForm = document.querySelector(".newsletter-form"); if (newsletterForm) { newsletterForm.addEventListener("submit", handleNewsletterSubmit); }

🔧 ملفات النظام

الملف الوصف الاستخدام
app.js الكود الرئيسي للموقع يحتوي على دالة handleNewsletterSubmit
newsletter-config.js إعدادات وإدارة النشرة الإعدادات وAdminPanel والقوالب
admin-newsletter.html لوحة التحكم إدارة المشتركين والإحصائيات

🚀 طريقة الاستخدام

1️⃣ للزائرين (الاشتراك)

✅ يمكن للزائرين الاشتراك عبر نموذج البريد في أسفل الموقع

2️⃣ للمسؤولين (لوحة التحكم)

اتبع الخطوات التالية للوصول إلى لوحة التحكم:

  1. افتح admin-newsletter.html في المتصفح
  2. ستظهر الإحصائيات والقوائم الكاملة
  3. يمكنك:
    • 👀 عرض قائمة المشتركين
    • 📊 الاطلاع على الإحصائيات
    • 📥 تصدير البيانات (CSV)
    • 🖨️ طباعة التقرير
    • 💾 إنشاء نسخة احتياطية
    • ✉️ إرسال بريد جماعي

💻 أوامر Console (للمطورين)

يمكنك استخدام الأوامر التالية في console المتصفح:

// عرض الإحصائيات newsletterAdmin.getStats(); // عرض قائمة المشتركين newsletterAdmin.getSubscribersList(); // تصدير CSV newsletterAdmin.exportAsCSV(); // طباعة التقرير newsletterAdmin.printReport(); // إنشاء نسخة احتياطية newsletterAdmin.createBackup(); // حذف مشترك newsletterAdmin.deleteSubscriber('email@example.com'); // إرسال بريد جماعي newsletterAdmin.sendBulkEmail('الموضوع', 'محتوى الرسالة');

🔐 الأمان والبيانات

ℹ️ البيانات تُخزن محلياً في localStorage على الجهاز:
  • لا يتم إرسال البيانات إلى خوادم غير آمنة
  • الاتصال بـ Formspree آمن (HTTPS)
  • يمكن حذف البيانات من إعدادات المتصفح

🐛 استكشاف الأخطاء

المشكلة: لا يظهر نموذج الاشتراك

✅ الحل: تأكد من أن newsletter-form موجود في HTML

المشكلة: الاشتراك لا يعمل

✅ الحل:

  1. تحقق من رمز Formspree في newsletter-config.js
  2. افتح console (F12) وتحقق من الأخطاء
  3. تأكد من تفعيل JavaScript في المتصفح

المشكلة: البيانات لا تظهر في لوحة التحكم

✅ الحل: قم بتحديث الصفحة أو استخدم أمر console:

// عرّض الكائن: window.newsletterAdmin = new NewsletterAdmin();
