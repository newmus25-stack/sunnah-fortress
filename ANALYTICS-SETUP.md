# 📊 دليل إعداد Google Analytics

## الخطوات:

### 1.اذهب إلى:
https://analytics.google.com/

### 2.سجّل دخول بحساب Google

### 3.أنشئ ملكية جديدة:
- اضغط على ⚙️ (إعدادات)
- اختر "إنشاء ملكية جديدة"
- املأ البيانات:
  - اسم الموقع: حصن السنة
  - عنوان الموقع: turath-cnt-sis.github.io
  - الفئة: موقع ويك，从事教育/参考

### 4.احصل على ID:
after creation, go to:
- Admin (⚙️)
- Data collection
- Copy the Measurement ID (look like: G-XXXXXXXX)
