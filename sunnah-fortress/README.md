# دليل نشر موقع "حصن السنة" مجاناً

تم تصميم هذا الموقع ليكون تطبيق صفحة واحدة (Single Page Application - SPA) لا يعتمد على خوادم معقدة، مما يجعله مثالياً للنشر المجاني السريع.

## طريقة النشر باستخدام GitHub Pages (موصى بها)

GitHub Pages توفر استضافة مجانية وموثوقة جداً للمشاريع الثابتة (مثل موقعنا هذا).

### الخطوات:

1. **إنشاء حساب:** 
   إذا لم يكن لديك حساب على GitHub، قم بإنشاء حساب مجاني على [github.com](https://github.com/).

2. **إنشاء مستودع جديد (Repository):**
   - اضغط على علامة `+` في أعلى يمين الشاشة واختر `New repository`.
   - اكتب اسم المستودع (مثلاً: `sunnah-fortress`).
   - اجعل المستودع عاماً `Public`.
   - لا تقم بتحديد خيار `Add a README file` (لأننا سنرفع الملفات الخاصة بنا).
   - اضغط على `Create repository`.

3. **رفع ملفات الموقع:**
   - بعد إنشاء المستودع، سترى شاشة بها خيارات، اضغط على الرابط `uploading an existing file`.
   - قم بسحب وإسقاط الملفات التالية (التي قمنا ببرمجتها) إلى المتصفح:
     - `index.html`
     - `styles.css`
     - `app.js`
     - `data.js`
   - في الأسفل عند "Commit changes"، اضغط على الزر الأخضر `Commit changes`.

4. **تفعيل GitHub Pages:**
   - من داخل المستودع، اذهب إلى الإعدادات `Settings` (في القائمة العلوية).
   - من القائمة الجانبية اليسرى، اضغط على `Pages`.
   - تحت قسم `Build and deployment`، ستجد خيار `Source`. تأكد من أنه `Deploy from a branch`.
   - تحت `Branch`، اختر `main` (أو `master`)، ثم اضغط `Save`.

5. **الحصول على الرابط:**
   - انتظر بضع دقائق (1-3 دقائق).
   - قم بتحديث الصفحة (`Pages`)، وسترى رسالة باللون الأخضر تقول: "Your site is live at [رابط الموقع]".
   - هذا الرابط هو الرابط الدائم لموقعك يمكنك مشاركته مع الجميع!

## كيفية تعديل المحتوى لاحقاً

نظراً لأننا استخدمنا `data.js` ليكون قاعدة بيانات مبسطة:
- لتعديل الشبهات أو إضافة شبهة جديدة، كل ما عليك فعله هو فتح ملف `data.js` بأي محرر نصوص.
- اتبع نفس النسق للبيانات الموجودة في المصفوفة `doubts`.
- بعد التعديل، إذا كنت تعمل محلياً، سيظهر التعديل مباشرة.
- إذا كنت قد رفعت الموقع على GitHub، اذهب إلى الملف `data.js` في مستودعك، واضغط على أيقونة القلم (Edit)، ضع التعديلات، ثم اضغط `Commit changes`. وسيتحدث الموقع تلقائياً.
